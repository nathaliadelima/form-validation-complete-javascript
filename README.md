# Complete Form Validation

Aplicação web desenvolvida para validação completa de formulário no front-end, utilizando JavaScript puro e expressões regulares (Regex).

## 🚀 Funcionalidades
- Validação de campos obrigatórios
- Verificação de tamanho mínimo do nome
- Validação de e-mail com Regex
- Validação de telefone com Regex
- Validação de senha segura (mínimo 8 caracteres, letra maiúscula e número)
- Bloqueio de envio com `preventDefault()`

## 🛠 Tecnologias
- HTML
- JavaScript
